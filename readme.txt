A useful tool for populating the block editor title and content.

== Installation ==

1. Go to 'Plugins > Add New' on your WordPress admin dashboard.
2. Search for 'Make Post Dirty' plugin from the official WordPress plugin repository.
3. Click 'Install Now' and then 'Activate'.

A useful tool for populating the block editor title and content.
